# BedHide Fabric 1.21.11

Command:
- `/bedhide hide` oder `/bedhide` – legt dein 3D-Spielermodell unter das nächste Bett
- `/bedhide show` – beendet das Versteck
- `/bedhide view` – Hinweis zur Kamera

## GitHub Build
Diese ZIP enthält bereits `.github/workflows/build.yml`.

Du musst nur den kompletten Inhalt dieses Ordners in dein GitHub Repository hochladen.
Danach startet unter **Actions** automatisch ein Build.
Die fertige JAR findest du im erfolgreichen Build unter **Artifacts** → `BedHide-Jar`.

Wichtig: Die Mod muss auf Server und allen Clients installiert sein.
